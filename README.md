# Vedanth Raja - Personal Portfolio

A premium, modern developer portfolio website designed with a dark, high-contrast aesthetic. Built using vanilla HTML5, custom CSS3, and JavaScript, it features a glowing cyber-neon style combined with polished micro-interactions and smooth animations.

![Design Preview](https://via.placeholder.com/1200x600/0a0d12/39FF14?text=Vedanth+Raja+Portfolio)

## 🚀 Features

- **Hacker/Developer Aesthetic**: Styled with a dark slate background (`#0a0d12`), neon green accents (`#39FF14`), and carbon-dark surfaces (`#111111`).
- **Drifting Particles Background**: A custom canvas-based particle system rendering 22 floating particles with subtle opacity pulsing (`0.06` to `0.24`) and upward drift.
- **Scroll Progress Bar**: A sleek `2px` neon green indicator at the top of the viewport that fills from left to right as you scroll down.
- **Dynamic Scrolling Marquee**: A dynamic loop translating marquee (`35s` infinite loop) containing `"THINK · DESIGN · BUILD · DELIVER · ALWAYS LEARNING · "` generated via JavaScript to ensure zero gaps on any screen width.
- **Staggered Scroll Animations**: Smooth fade-up entrance transitions (`0.6s ease`) triggered using an `IntersectionObserver` (0.15 threshold), staggering section children by `0.1s` and skills pills by `0.08s`.
- **Interactive VR Logo**: A customized logo with an underline green gradient and smooth-scroll-to-top interaction.
- **Terminal Card Details**: An interactive Command Line terminal simulation containing biographical facts with sentence-cased keys and polished layouts.
- **Copy-to-Clipboard Networking**: Fast one-click copy functionality on the email address showing visual confirmation indicators.
- **Custom Gold Vector Favicon**: A responsive and glowing SVG icon showing a stylized cutout pattern against a rich gold gradient square base.

## 🛠️ Built With

- **HTML5**: Semantic tags layout structure.
- **CSS3 (Custom Styling)**: Advanced styles, custom grid layouts, custom properties variables, animation keyframes, and typography sizes.
- **JavaScript (ES6)**: Canvas rendering, scroll tracking, intersection observers, dynamic string generation, and clipboard access.
- **Typography & Fonts**:
  - [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono): logo, section headings, nav links, labels, pills, and terminal commands.
  - [Inter](https://fonts.google.com/specimen/Inter): body text, paragraphs, and project cards descriptions.
