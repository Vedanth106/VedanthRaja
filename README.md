# Vedanth Raja - Personal Portfolio

A sleek, minimal, "brutalist-meets-hacker" personal portfolio website built with pure HTML, CSS, and JavaScript.

![Design Preview](https://via.placeholder.com/1200x600/0a0a0a/39FF14?text=Vedanth+Raja+Portfolio)

## 🚀 Features

- **Hacker Aesthetic**: High-contrast black background with neon green accents.
- **Typewriter Effect**: Dynamic hero section with a cycling typewriter animation.
- **Terminal Card**: Interactive looking "About" section styled as a command-line terminal.
- **Project Grid**: Minimalist cards with neon borders and glow hover effects.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Smooth Animations**: Subtle fade-in effects using the Intersection Observer API.
- **Copyable Email**: One-click email copying for quick networking.

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **Vanilla CSS3**: Advanced styling, animations, and custom typography.
- **JavaScript**: Typewriter logic, scroll observers, and copy-to-clipboard functionality.
- **Fonts**: [JetBrains Mono](https://fonts.google.com/specimen/JetBrains+Mono) for headers, [Inter](https://fonts.google.com/specimen/Inter) for body text.
